# Brand Assets

This repo keeps the Meridian brand asset set under [`docs/assets/`](./assets/)
for repository presentation, social preview, release materials, and documentation.

## Included Assets

| File | Dimensions | Description |
|------|-----------|-------------|
| `meridian_mark_flat.svg` | 1024x1024 viewBox | Angular warrior mark silhouette. Responds to `prefers-color-scheme`. |
| `meridian_wordmark_flat.svg` | 720x128 viewBox | "Meridian" in Inter 700 + "Constitutional Kernel" subtitle. Light/dark responsive. |
| `meridian_lockup_flat.svg` | 960x256 viewBox | Warrior mark + wordmark combined. Preferred for README headers. |
| `logo_favicon_64.png` | 64x64 | White warrior mark on #050508 dark background. Browser tab icon. |
| `logo_avatar_192.png` | 192x192 | White warrior mark on #050508. GitHub/social avatar. |
| `logo_banner_1200x630.jpg` | 1200x630 | Social preview / OG share image. Warrior mark on dark field. |

## Usage

- Prefer the flat SVG assets for README, docs, and UI surfaces.
- Use the raster PNG/JPG assets only where an image fallback or social banner is needed.
- Do not stretch, recolor, or crop asymmetrically.
- Do not use the banner image inline as hero art in technical docs.
- All SVG assets support `prefers-color-scheme` for automatic light/dark adaptation.
