# Support

## Getting Help

### Bugs and Feature Requests

Open an issue on GitHub using the appropriate template:
- [Bug report](../../issues/new?template=bug_report.yml)
- [Feature request](../../issues/new?template=feature_request.yml)

### Questions and Discussion

Use [GitHub Discussions](../../discussions) for questions, ideas, and
general conversation about the project.

### Security Issues

**Do not open public issues for security vulnerabilities.** See
[SECURITY.md](SECURITY.md) for reporting instructions.

## Response Time

The maintainer targets 48-hour first-touch triage for issues and pull
requests. Complex issues may take longer to resolve, but you should
expect an initial acknowledgment within two business days.

## What We Can Help With

- Kernel primitive usage and configuration
- Architecture questions
- Integration with agent runtimes
- Bug investigation and fixes
- Feature design discussion

## What Is Out of Scope

- Support for the hosted Meridian service (separate from this kernel)
- General AI/ML questions unrelated to governance
- Custom deployment consulting (consider sponsoring for priority support)
